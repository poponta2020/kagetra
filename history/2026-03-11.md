## 完了した作業

### 1. LINE添付画像の即時削除バグ修正
- **問題**: `DocumentConverter.cleanup` を LINE送信直後に呼んでいたため、LINEサーバーが画像を取得する前にファイルが消えていた
- **修正**: `bin/send_notifications.rb` の cleanup呼び出しを削除し、バッチ冒頭で「1日以上前の line_tmp を削除する `cleanup_old_line_tmp`」に変更
- **ファイル**: `bin/send_notifications.rb`

### 2. T-01 LINEメッセージ文言変更
- 「詳細は景虎の方をご覧ください」→「大会要項は添付画像もしくは景虎でご確認ください」
- **ファイル**: `bin/send_notifications.rb`

### 3. admin before フィルターの nil チェック追加
- **問題**: `@user.admin` を nil チェックなしで呼んでいたため、未ログイン時に500エラー
- **修正**: `halt 403 unless @user && @user.admin`
- **ファイル**: `routes/admin.rb`

### 4. LINE公開エンドポイントの403修正
- **問題**: `routes/misc.rb` の `get '/public/*'` が `/public/line_tmp/...` を横取りして403を返していた
- **修正**: `filter_public` に `/line_tmp/` パスの早期リターン処理を追加（パストラバーサル対策付き）
- **ファイル**: `routes/misc.rb`

### 5. WordファイルのLINE送信動作確認
- `.docx` → LibreOffice → PDF → RMagick → JPG の変換が正常動作することを本番サーバーで確認
- 4ページの変換成功を確認

### 6. バッチ通知の本番テスト実行
- `bundle exec ruby bin/send_notifications.rb` を本番で実行
- A級へのテキスト通知・画像通知ともに正常動作確認

### 7. 申込フロー画面の復旧（Docker環境）
- **問題1**: Docker内の `conf.rb` が古く「申込管理」メニューが未追加 → `conf.rb` 削除・再起動で解決
- **問題2**: DBマイグレーション v11 未実行（テーブルなし） → マイグレーション実行
- **問題3**: `application_flow.haml` に `#templ-flow-detail`・`#templ-participants`・`#container-flow-detail` が欠落 → 追加
